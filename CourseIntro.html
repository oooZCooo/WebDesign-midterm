Basic Introduction of EGR309:

This course is a programming and engineering course. Students will use various machine part and Arduino board to build a simple machine and program it. Student will face different interesting challenge like build a machine to play piano tiles.

Requiament for this course:

  • Have basic understanding about Java
  • Understanding principles of physics about Circuit
  • Know how to uses Arduino board
